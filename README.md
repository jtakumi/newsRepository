# newsRepository

技術のことからエンタメまで、日々のニュースや学習メモを日本語でまとめているリポジトリです。

## カテゴリ

| カテゴリ | ディレクトリ | 内容 |
| --- | --- | --- |
| AI | [`AI/`](./AI) | フロンティアモデル、AIインフラ、開発者向けツールなどの動向 |
| 主要ニュース | [`major-news/`](./major-news) | 日本国内および国際の主要ニュースのブリーフィング |
| VTuber | [`vtuber/`](./vtuber) | VTuber界隈のライブ・音楽・企業案件などのトピック |
| 英語表現 | [`english-expressions/`](./english-expressions) | 日本語表現を自然な英語に言い換える学習メモ |

## ディレクトリ構成

各カテゴリは `YYYY/MM/` の年月フォルダで整理しています。

```
<category>/YYYY/MM/YYYY-MM-DD.md
```

例:

- `AI/2026/06/2026-06-27.md`
- `major-news/2026/06/2026-06-27.md`
- `vtuber/2026/06/2026-06-27.md`
- `english-expressions/2026/06/2026-06-27-otsukaresama.md`

英語表現カテゴリは、1日1テーマでファイル名末尾にトピックを付与する場合があります。

## 方針

- 主要事項に絞り、炎上・ゴシップは原則として扱いません。
