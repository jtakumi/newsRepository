# 英語表現ミニ講座 2026-07-16

## 今日の表現

**差し戻す**

仕事では、レビューや申請をそのまま承認せず、修正のために相手へ戻すときに使います。

## そのまま直訳すると不自然な理由

「差し戻す」を機械的に **return** とだけ訳すと、物理的に返却するのか、申請を却下するのか、修正を求めて戻すのかが分かりません。

英語では、何を目的に戻すのかを動詞や補足で明確にします。

## 自然な英語表現

### send it back for revisions

修正を求めて差し戻す、という意味を最も明確に伝えられます。

> I'll send this back for revisions.

### request changes

コードレビューや文書レビューで、変更を求めるときに自然です。GitHubのレビュー操作にも使われる表現です。

> I requested changes on the pull request.

### return it with comments

コメントを付けて返すことを強調したいときに使えます。

> I'll return the draft with a few comments.

### reject the request

修正前提ではなく、申請自体を却下するときに使います。**send back for revisions** より強い表現です。

> The request was rejected because required information was missing.

## 場面別の言い換え

### エンジニアリング

- **I requested changes on the PR.**
- **I'm sending this back because the error handling is incomplete.**
- **Could you address these comments and resubmit it?**

### ビジネス

- **I'll send the proposal back for revisions.**
- **The application was returned because some fields were incomplete.**
- **Please revise the document and submit it again.**

### 日常会話

日常会話では「差し戻す」を使う場面は少なく、具体的に言う方が自然です。

- **They asked me to redo it.**
- **I had to submit it again.**

## 短い例文

> The reviewer sent the document back for revisions.  
> レビュアーは文書を修正のため差し戻しました。

> I requested changes because the test cases were missing.  
> テストケースが不足していたため、変更を依頼しました。

> Could you fix these points and resubmit the form?  
> これらの点を修正して、フォームを再提出してもらえますか。

## 今日の5分練習

1. 自分が最近レビューしたPRや文書を1つ思い出します。
2. 差し戻した理由を英語で1文にします。
3. 次の型を使って声に出します。

> I'm sending this back because ______.

4. 次に、相手への依頼を加えます。

> Could you ______ and resubmit it?

例:

> I'm sending this back because the loading-state handling is incomplete. Could you add the missing case and resubmit it?
